🦸‍♂️ Classificador de Nível de Herói

📖 Sobre o Projeto

Este projeto foi desenvolvido como desafio prático para treinar lógica de programação em JavaScript.

O sistema classifica o nível de um herói com base na sua quantidade de experiência (XP), utilizando funções e estruturas condicionais.

🧠 Conceitos Aplicados

✔️ Declaração de variáveis

✔️ Estruturas condicionais (if / else if / else)

✔️ Funções com retorno (return)

✔️ Template strings

✔️ Organização de lógica em blocos reutilizáveis

🎯 Objetivo

Criar uma função chamada calcularNivel que:

Recebe a quantidade de XP como parâmetro.

Retorna o nível correspondente.

Exibe no console a mensagem final com o nome do herói e seu nível.

🧠 Conceitos Aplicados

Variáveis (let)

Funções com parâmetro

Estruturas condicionais (if / else if / else)

Operadores lógicos (&&)

Template Strings

Organização de código

🏆 Regras de Classificação
XP	Nível

≤ 1000	Ferro

1001 – 2000	Bronze

2001 – 5000	Prata

5001 – 7000	Ouro

7001 – 8000	Platina

8001 – 9000	Ascendente

9001 – 10000 Imortal

> 10000	Radiante
